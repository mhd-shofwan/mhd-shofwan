## 🚀 Public Projects

### [SPSB - Student Admission System Showcase](https://github.com/mhd-shofwan/spsb-showcase)

A modern student admission management system built for Islamic boarding schools. SPSB streamlines the entire enrollment process through a guided registration workflow and provides administrators with powerful tools for managing applicants, payments, academic years, and analytical reports.

Features include multi-step registration, digital signatures, payment tracking, WhatsApp integration, Excel reporting, and interactive dashboards.

> The production source code remains private. This repository serves as a public showcase and documentation.

---

### [Absensi Digital (Kajian)](https://github.com/mhd-shofwan/absensi-digital)

A lightweight, real-time, kiosk-based attendance system designed specifically for Islamic study circles (Kajian). Built with Vanilla JavaScript, Tailwind CSS, and Firebase.

Features include kiosk mode for participants, device approval management, real-time dashboards, and automated Excel reporting.

---

### [MySQL Repair Tools for XAMPP](https://github.com/mhd-shofwan/mysql-repair-tools)

A simple utility based on batch scripts that helps resolve common MySQL issues in XAMPP environments.

The project is open source and aims to provide a quick and practical solution for recovering broken MySQL installations.
